## FFmpeg-Wails v0.0.1

跨平台 FFmpeg 圖形化工具，基於 Wails (Go) + Nuxt 4 (Vue 3)。

### ✨ 功能

- 🎬 **影片/音訊轉檔** — H.264/H.265/VP9/AV1, AAC/MP3/Opus, CRF/Bitrate 控制
- 📡 **RTMP/SRT 推流** — YouTube/Twitch 一鍵 preset，支援即時來源
- 📋 **任務管理** — 即時進度、FPS/Bitrate/Speed 顯示、Log 檢視
- 🎨 **深色 UI** — UnoCSS + Vue 3 響應式介面

### 📦 下載

| 檔案 | 平台 |
|------|------|
| `FFmpeg-Wails-windows-amd64.exe` | Windows x64 |
| `FFmpeg-Wails-darwin-amd64` | macOS Intel |
| `FFmpeg-Wails-darwin-arm64` | macOS Apple Silicon |
| `FFmpeg-Wails-linux-amd64` | Linux x64 |

### 🚀 使用方式

1. 確保已安裝 [FFmpeg](https://ffmpeg.org/)
2. 下載對應平台的二進位檔案
3. 執行即可

### 🧱 技術棧

Go (Wails v2) + Vue 3 (Nuxt 4) + UnoCSS
